# v0.7 Log Search Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `--grep`, `--since`, `--context`, and `--json` flags to the `logs` command with server-side follow-mode filtering and TUI regex toggle.

**Architecture:** Non-follow filtering runs client-side using `DiskLogReader` (moved to a shared module). Follow-mode `--grep` runs daemon-side in the broadcast subscriber. `--since restart` uses durable restart markers written synchronously in `spawn_process()` before either capture task starts. The shared `AtomicU64` sequence counter enables cross-stream boundary resolution.

**Tech Stack:** Rust, tokio, clap 4, serde, regex, ratatui

**Spec:** `docs/superpowers/specs/2026-03-18-v0.7-log-search-design.md`

```rust
/// Pattern matching mode for log line filtering.
pub enum MatchMode {
    /// Case-sensitive substring match.
    Substring(String),
    /// Regex match.
    Regex(regex::Regex),
}

impl MatchMode {
    pub fn is_match(&self, line: &str) -> bool {
        match self {
            MatchMode::Substring(pat) => line.contains(pat.as_str()),
            MatchMode::Regex(re) => re.is_match(line),
        }
    }
}
```

Add new scan methods that accept `MatchMode`:

```rust
pub fn scan_matching_lines_mode(&mut self, mode: &MatchMode, stream: StreamMode) -> Vec<usize> {
    self.scan_matching_lines_mode_from(mode, stream, 0)
}

pub fn scan_matching_lines_mode_from(
    &mut self,
    mode: &MatchMode,
    stream: StreamMode,
    start_from: usize,
) -> Vec<usize> {
    match stream {
        StreamMode::Stdout => self.scan_single_stream_mode(mode, LineSource::Stdout, start_from),
        StreamMode::Stderr => self.scan_single_stream_mode(mode, LineSource::Stderr, start_from),
        StreamMode::Both => self.scan_interleaved_mode(mode, start_from),
    }
}
```

Add internal helpers `scan_single_stream_mode` and `scan_interleaved_mode` (same as existing `scan_single_stream`/`scan_interleaved` but using `mode.is_match()` instead of `line.contains(filter)`).

```rust
/// Result of the filter pipeline for a single process.
pub struct SearchResult {
    pub process: String,
    pub lines: Vec<SearchLine>,
}

/// A single line in search results.
pub struct SearchLine {
    pub line_number: usize,
    pub source: LineSource,
    pub text: String,
    pub is_context: bool,

/// Parameters for the filter pipeline.
pub struct SearchParams {
    pub grep: Option<String>,
    pub regex: bool,
    pub since: Option<String>,
    pub tail: usize,
    pub context: Option<u32>,
    pub stderr: bool,
}

impl SearchParams {
    /// Build a MatchMode from grep + regex flags. Returns None if no grep.
    pub fn match_mode(&self) -> Result<Option<MatchMode>, String> {
        match &self.grep {
            None => Ok(None),
            Some(pattern) => {
                if self.regex {
                    let re = Regex::new(pattern)
                        .map_err(|e| format!("invalid regex: {}", e))?;
                    Ok(Some(MatchMode::Regex(re)))
                } else {
                    Ok(Some(MatchMode::Substring(pattern.clone())))
                }
            }
        }
    }

    fn stream_mode(&self) -> StreamMode {
        if self.stderr {
            StreamMode::Stderr
        } else {
            StreamMode::Stdout
        }
    }
}

/// Run the filter pipeline for a single process.
pub fn search_process(
    reader: &mut DiskLogReader,
    process: &str,
    params: &SearchParams,
    uptime_secs: Option<u64>,
) -> Result<SearchResult, String> {
    let mode = params.stream_mode();
    let match_mode = params.match_mode()?;

    // Step 1: Resolve --since to starting line
    let start_line = resolve_since(reader, &params.since, mode, uptime_secs)?;

    // Step 2: Find matching lines
    let total = line_count_for_mode(reader, mode);
    let matching = if let Some(ref mm) = match_mode {
        reader.scan_matching_lines_mode_from(mm, mode, start_line)
    } else {
        // No grep — all lines in range
        (start_line..total).collect()
    };

    // Step 3: Apply --tail windowing
    let windowed: Vec<usize> = if matching.len() > params.tail {
        matching[matching.len() - params.tail..].to_vec()
    } else {
        matching
    };

    // Step 4: Expand context
    let expanded = expand_context(&windowed, params.context, total);

    // Step 5: Read lines
    let match_set: std::collections::HashSet<usize> = windowed.iter().copied().collect();
    let lines = read_result_lines(reader, process, mode, &expanded, &match_set);

    Ok(SearchResult {
        process: process.to_string(),
        lines,
    })
}
```

```rust
/// Resolve --since spec to a starting line number.
fn resolve_since(
    reader: &mut DiskLogReader,
    since: &Option<String>,
    mode: StreamMode,
    uptime_secs: Option<u64>,
) -> Result<usize, String> {
    let Some(spec) = since else { return Ok(0) };

    match spec.as_str() {
        "start" => Ok(0),
        "restart" => resolve_since_restart(reader, mode),
        s => resolve_since_duration(reader, s, mode, uptime_secs),
    }
}

fn resolve_since_restart(
    reader: &mut DiskLogReader,
    mode: StreamMode,
) -> Result<usize, String> {
    // Scan stdout backward for restart marker
    let stdout_total = reader.line_count(LineSource::Stdout);
    if stdout_total == 0 {
        return Ok(0);
    }

    // Scan backward in chunks
    const CHUNK: usize = 500;
    let mut offset = stdout_total;
    while offset > 0 {
        let start = offset.saturating_sub(CHUNK);
        let count = offset - start;
        if let Ok(lines) = reader.read_lines(LineSource::Stdout, start, count) {
            for (i, line) in lines.iter().enumerate().rev() {
                if line.starts_with("[agent-procs] Restarted")
                    || line.starts_with("[agent-procs] File changed, restarted")
                {
                    let marker_line = start + i;
                    let marker_line_after = marker_line + 1;

                    if mode == StreamMode::Stdout || mode == StreamMode::Both {
                        return Ok(marker_line_after);
                    }

                    // Cross-stream: find stderr position via sequence number
                    return resolve_cross_stream_boundary(reader, marker_line);
                }
            }
        }
        offset = start;
    }

    // No marker found — serve everything
    Ok(0)
}

fn resolve_cross_stream_boundary(
    reader: &mut DiskLogReader,
    stdout_marker_line: usize,
) -> Result<usize, String> {
    // Get the marker's sequence number from stdout LIDX
    // Then find first stderr line with seq >= that value
    // This requires new DiskLogReader methods (added in Task 8)
    reader.find_stderr_position_by_stdout_seq(stdout_marker_line)
        .map_err(|e| format!("cross-stream resolution failed: {}", e))
}

fn resolve_since_duration(
    reader: &mut DiskLogReader,
    spec: &str,
    mode: StreamMode,
    uptime_secs: Option<u64>,
) -> Result<usize, String> {
    let duration_secs = parse_duration(spec)?;

    // Find incarnation boundary
    let incarnation_start = resolve_since_restart(reader, mode).unwrap_or(0);
    let total = line_count_for_mode(reader, mode);
    let incarnation_lines = total - incarnation_start;

    let Some(uptime) = uptime_secs else {
        eprintln!("warning: --since duration is approximate for exited processes");
        return Ok(incarnation_start);
    };

    if uptime <= duration_secs {
        return Ok(incarnation_start);
    }

    let estimated = incarnation_start
        + ((incarnation_lines as f64) * (1.0 - duration_secs as f64 / uptime as f64)) as usize;
    Ok(estimated.min(total))
}

fn parse_duration(spec: &str) -> Result<u64, String> {
    let (num_str, multiplier) = if let Some(n) = spec.strip_suffix('s') {
        (n, 1u64)
    } else if let Some(n) = spec.strip_suffix('m') {
        (n, 60)
    } else if let Some(n) = spec.strip_suffix('h') {
        (n, 3600)
    } else {
        return Err(format!("invalid --since spec: '{}'. Use Ns, Nm, Nh, 'restart', or 'start'", spec));
    };
    let n: u64 = num_str.parse().map_err(|_| format!("invalid number in --since: '{}'", spec))?;
    Ok(n * multiplier)
}
```

```rust
/// Expand match line numbers with N lines of context. Merge overlapping ranges.
fn expand_context(
    matches: &[usize],
    context: Option<u32>,
    total_lines: usize,
) -> Vec<(usize, bool)> {
    let Some(n) = context else {
        // No context — just return matches tagged as non-context
        return matches.iter().map(|&ln| (ln, false)).collect();
    };
    let n = n as usize;

    if matches.is_empty() {
        return Vec::new();
    }

    let match_set: std::collections::HashSet<usize> = matches.iter().copied().collect();

    // Build ranges [match-N, match+N] for each match, merge overlapping
    let mut ranges: Vec<(usize, usize)> = Vec::new();
    for &m in matches {
        let start = m.saturating_sub(n);
        let end = (m + n + 1).min(total_lines);
        if let Some(last) = ranges.last_mut() {
            if start <= last.1 {
                last.1 = last.1.max(end); // merge
                continue;
            }
        }
        ranges.push((start, end));
    }

    // Flatten ranges into (line_number, is_context) pairs
    let mut result = Vec::new();
    for (start, end) in ranges {
        for ln in start..end {
            let is_context = !match_set.contains(&ln);
            result.push((ln, is_context));
        }
    }
    result
}
```

```rust
/// Given a stdout line index, find the corresponding stderr line index
/// by looking up the stdout line's sequence number and finding the first
/// stderr line with sequence >= that value.
pub fn find_stderr_position_by_stdout_seq(
    &mut self,
    stdout_line: usize,
) -> io::Result<usize> {
    // Get sequence number of the stdout line
    let stdout_seq = self.get_sequence_number(LineSource::Stdout, stdout_line)?;

    // Binary search stderr LIDX for first line with seq >= stdout_seq
    let stderr_total = self.line_count(LineSource::Stderr);
    for i in 0..stderr_total {
        if let Ok(seq) = self.get_sequence_number(LineSource::Stderr, i) {
            if seq >= stdout_seq {
                return Ok(i);
            }
        }
    }
    Ok(stderr_total) // all stderr is before the boundary
}

/// Get the LIDX sequence number for a specific line.
fn get_sequence_number(&mut self, source: LineSource, line: usize) -> io::Result<u64> {
    // Navigate to the right segment and read the LIDX record
    let segments = self.segments(source);
    let mut cumulative = 0;
    for seg in segments {
        if line < cumulative + seg.line_count {
            let local_line = line - cumulative;
            let mut reader = IndexReader::open(&seg.idx_path)?
                .ok_or_else(|| io::Error::new(io::ErrorKind::NotFound, "no index"))?;
            let records = reader.read_range(local_line, 1)?;
            return Ok(records[0].seq);
        }
        cumulative += seg.line_count;
    }
    Err(io::Error::new(io::ErrorKind::InvalidInput, "line out of range"))
}
```
